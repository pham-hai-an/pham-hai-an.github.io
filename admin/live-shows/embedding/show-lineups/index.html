Show Lineups

Purpose of Show Lineups​

When you create a new live show, you will have to replace the previous show’s embed code with a new one if you are using the same landing page or entry point.

A show lineup will be able to reduce your effort needed in replacing embed codes. By grouping several live shows into one lineup, a lineup embed code will be generated. This lineup embed code can be used such that when you switch from one show to the next, you will not need to change embed codes.

How To Create A Show Lineup​

A typical use case of using a show lineup is to organise your shows based on entry points.

For example, you can create a lineup for shows that will appear in the main real estate of your landing page, and another lineup for shows that will appear in a different webpage.

To create a show lineup:

  1. Create your live shows as per normal.
  2. Click on the Show Lineups in the Live Show listing page.
  3. Click on Create Lineup
  4. Give the lineup a title for your own reference (e.g Landing Page shows). Viewers will not see this title.
  5. Select the shows that you have created to add into the lineup and sort in order of appearance.
  6. Select the Featured Show.
    • This is selecting what the viewers should see if there is no ongoing live show from the lineup.
    • By default, if no live shows from the lineup has started at all, viewers will see the pre-stream cover image of the first live show.
  7. Save the lineup. A lineup embed code will be generated.
  8. Copy the embed code and paste it onto the HTML script of your landing page.

On the viewer’s side, watching a show from a lineup is no different from watching a show not in a lineup. You can start a live show from a lineup as per normal.

Order Of Shows In A Lineup​

The order in which your live show is sorted is the order which the Featured Show will come into play. However, you are not restricted to starting live shows in the exact same order.

note

Show Lineups do not support concurrent live streaming. If a second live show in a lineup is started by a different host, viewers will not be able to enter the first live show anymore.
